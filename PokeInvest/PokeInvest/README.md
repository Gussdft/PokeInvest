# PokeInvest

## Clé RapidAPI

Pour profiter des prix Cardmarket fournis par l'API Pokémon TCG disponible sur RapidAPI, ajoutez une entrée `RAPID_API_KEY` dans l'`Info.plist` de l'app avec votre clé. Sans cette clé, l'application utilisera automatiquement le backend TCGDex (sans cotations) afin de continuer à afficher les cartes.
